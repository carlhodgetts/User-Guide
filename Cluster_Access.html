Cluster Access

Architecture

The cluster is a high performance computational infrastructure which is comprised of a cluster of individual computers or nodes. Each node serves a specific purpose and in our architecture there are 3 different types of nodes namely, login nodes, master node and worker nodes. In the RHUL cluster, there are 2 login nodes and as the name suggests the login nodes' function is to log on the user to the cluster. There is 1 master node for the cluster which distributes and allocates computational jobs to the 4 worker nodes in the cluster.

Access through psyclogin

The psyclogin node is to be used for cluster access if it is required for GUI (Graphical User Interface) tools on the cluster. This node is provided with high-performance graphics drivers for this. psyclogin can be accessed through Microsoft Windows Remote Desktop Server. This Remote Desktop Server can be accessed directly while operating from a local PC on campus and for off-campus access is granted only through VPN (details of VPN access is provided here). After connecting to the VPN use the Remote Desktop Connection software (Windows RDP / Ubuntu Remmina) and connect to the Microsoft Windows Remote Desktop Server( server address: prod-mrirdp01.rhul.ac.uk ). Use your college ID and password and domain as cc to log on to the WIndows Server. After sucessfully logging on to the RDP server you will find the VNCViewer application for accessing the cluster with GUI and putty application for logging on with command line interface (CLI) using Secure Shell (SSH).

VNCViewer: Open the VNCViewer app and enter psyclogin.rhul.ac.uk:[VNC_session_number] in the VNC server dropdown box. The [VNC_session_number] is an unique session code that you should have received from the IT team (If you haven't been provided one, please email the psychology IT Team here). After clicking Connect enter your VNC password and log-on to the cluster.

Putty: Open the Putty app and enter psyclogin.rhul.ac.uk in the Host Name field. Check if the port is selected to be [22] and the connection type is SSH. After clicking Open enter your college ID in the login as: field and hit Return to be prompted for your password. Once you have entered your password and hit the Return key, It will log you on to the cluster through the CLI (Remember that for additional security, the cursor doesn't move while entering your password in the command prompt).

Create VNC Session: A [VNC_session_number] can be created upon logging on to psyclogin.rhul.ac.uk through putty. From the CLI use the command vncserver and it will create a [VNC_session_number]. This command is to be used only if you haven't had a VNC session number before, otherwise you can use the previously created session ID.

Access through psyclogin02

This login node can be accessed through the Command Line Interface (CLI) directly while operating from a local PC on campus and through VPN for off-campus use. Upon opening the Terminal (CMD prompt Windows) please issue the following command to access the node.
ssh[college_ID]@psyclogin02.rhul.ac.uk. When prompted enter your password and hit the Return key to log you on to the cluster through the CLI.

Use of Storage on Cluster

Generally the workspace/folder of individual users on the cluster are created under the workspace of individual PIs.
